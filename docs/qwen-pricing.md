## 备注

- 阶梯计价模型：价格按单次请求输入 Token 量分档，该请求所有 Token 均按对应档价格结算
- 思考模式输出价格包含思维链 + 最终回答
- Batch 调用可享 50% 折扣
- 上下文缓存仅对输入 Token 有折扣，与 Batch 不能同时生效
- 所有模型均支持中国内地部署，部分支持海外部署（弗吉尼亚、新加坡、法兰克福、东京）
